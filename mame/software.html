Computer

Nothing too exciting here. Gateway i5 system. Arcade VGA 5000 video card.

Software

This is a list of the software I'm using in the cabinet:

  • Arch Linux GroovyArcade uses Arch Linux as its base operating system. I don't particularly like Arch, but that's what it is.
  • M.A.M.E. is the arcade emulator that allows all of this retro magic, it emulates over 3000 different video games, from the earliest in the late 70s to the NeoGeo games of a few years ago. An amazing feat of programming. GroovyMame is set of patches for MAME specifically for arcade monitors. The difference is stunning. All games run at their native resolutions, etc.
  • My own frontend which does some things I wanted. Written using pygame.
  • Daphne For playing Dragon's Lair and Space Ace laserdisc games.
  • Vim for editing text and config files.
